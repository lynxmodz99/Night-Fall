## 📥 Installation

### 1. Clone the Repository OR Download It

### 2. Install Python Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install FFmpeg (Required)

For voice-related features, FFmpeg is required.

**Official FFmpeg Website:**
https://ffmpeg.org/download.html

**Recommended Windows Build:**
https://www.gyan.dev/ffmpeg/builds/

### 📌 FFmpeg Setup

1. Open the Gyan FFmpeg Builds page.
2. Download the latest **`ffmpeg-release-essentials.zip`**.
3. Extract the archive.
4. Open the extracted folder and navigate to the `bin` directory.
5. Copy **`ffmpeg.exe`**, **`ffplay.exe`**, and **`ffprobe.exe`**.
6. Paste these files into the **root folder** of this project (where `main.py` is located).

### 4. Run the Project

```bash
python main.py
```
